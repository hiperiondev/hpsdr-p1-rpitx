# hpsdr-p1-rpitx

Transmitter with Raspberry PI over OpenHPSDR

*COMPILE PROJECT:*
- git clone --recurse-submodules https://github.com/hiperiondev/hpsdr-p1-rpitx
- cd hpsdr-p1-rpitx/build
- make

*USE:*
hpsdr-p1-rpitx [flags]

flags:
- -atlas:  
- -hermes: 
- -griffin: 
- -angelia: 
- -orion: 
- -orion2: 
- -hermeslite:
- -hermeslite2: 
- -c25 :
- -diversity:  
- -P1:
- -P2:
- -debug:
- -iqburst:
